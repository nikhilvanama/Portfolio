---
description: How to update the portfolio screenshots for the README
---

1. Start a local server in the project directory.
// turbo
2. Run `python -m http.server 8181` (or use any other port).
3. Use the browser to navigate to `http://localhost:8181`.
4. Navigate to each section (About, Skills, Journies, Projects, Contact).
5. Capture a full-page screenshot for each.
6. Save and overwrite the images in `n:\Portfolios\Portfolio\assets\images\screenshots/`:
    - `about_section.png`
    - `skills_section.png`
    - `journies_section.png`
    - `projects_section.png`
    - `contact_section.png`
7. Verify that the `README.md` correctly displays the new visuals.

> [!TIP]
> This workflow ensures your documentation always stays in sync with your latest UI changes.
